# Multi-Room Thermostat Add-on Repository

This repository contains the Multi-Room Thermostat add-on for Home Assistant.

## Installation

1. Add this repository to Home Assistant:
   - Go to Supervisor → Add-on Store
   - Click the three dots (⋮) in the top right
   - Select "Repositories"
   - Add repository URL: `https://github.com/yourusername/MultiStat`
   - Click "Add"

2. Install the "Multi-Room Thermostat" add-on from the store

3. Configure and start the add-on

For detailed documentation, see the [add-on README](multistat/README.md).
